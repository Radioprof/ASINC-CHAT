Client module documentation
=================================================

Клиентское приложение для обмена сообщениями. Поддерживает
отправку сообщений пользователям, которые сейчас подключены, сообщения шифруются.


Аргументы командной строки:

``python client.py {имя сервера} {порт} -n или --name {имя пользователя} -p или -password {пароль}``

1. {имя сервера} - сетевой адрес сервера сообщений.
2. {порт} - порт сервера, по которому принимаются подключения
3. -n или --name - имя пользователя для входа в систему.
4. -p или --password - пароль пользователя.

Аргументы можно не указывать. Имя пользователя и пароль указывают  только совместно.

Примеры использования:

* ``python client.py``

*Запуск приложения с параметрами по умолчанию.*

* ``python client.py 192.168.1.1 10000``

*Запуск приложения с указанием подключаться к серверу по адресу 192.168.1.1:10000*

* ``python -n test1 -p 123``

*Запуск приложения с пользователем test1 и паролем 123*


client.py
~~~~~~~~~

Запускаемый модуль содержит парсер аргументов командной строки и функционал инициализации приложения.

client. **arg_parser** ()
    Парсер аргументов командной строки, возвращает кортеж из 4 элементов:
    
	* адрес сервера
	* порт сервера
	* имя пользователя
	* пароль пользователя
	
    Выполняет проверку на корректность номера порта.
